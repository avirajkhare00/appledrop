# appledrop

A simple multiplayer game. Try the game [here](http://appledrop.codeignition.tech/).

## Quickstart

- `git clone` this project
- `npm install`
- `npm start`

## Requirements

[Node.js](https://nodejs.org) is required to install dependencies and run scripts via `npm`.
